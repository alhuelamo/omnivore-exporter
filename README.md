# omnivore-exporter

Utility to export articles from [Omnivore](https://omnivore.app/).

## Prerequisites

- [Scala CLI](https://scala-cli.virtuslab.org/)

## IDE setup

```shell
scala-cli setup-ide .
```

## Running tests

```shell
scala-cli test .
```
